# 💎 Diamondwrt: Exclusive Custom Firmware

**Diamondwrt** は、特定ユーザーの高度な要件に合わせて極限までチューニングされた、完全オーダーメイドの専用ファームウェアです。

ベースシステムには強力なオープンソースプロジェクトである OpenWrt およびその優れた派生リポジトリを採用し、最新のネットワーク機能と高い安定性を確保しています。

⚠️ **重要なお知らせ / Notice**
本リポジトリでビルドされるリリースファイルには、独自の強固な暗号化処理が施されています。これらは許可された専用環境でのみ稼働するよう設計されたプライベートビルドであり、**復号キーを持たない一般のユーザーがフラッシュ（インストール）して使用することはできません。**あらかじめご了承ください。
